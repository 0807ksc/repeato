# Archived Work Documents

## 목적
- GitHub issue 또는 다른 기준 문서로 대체되어 더 이상 활성 참조가 필요하지 않은 로컬 문서를 보관한다.

## 이동 기준
- 이슈/PR이 canonical source가 되었고 로컬 문서가 작업 중 직접 참조되지 않는다.
- 역사 보존 가치는 있지만 `doc/work/` 루트에 둘 만큼 활성도가 높지 않다.
- 완료된 회의 메모, 종료된 spec 초안, 과거 snapshot 중 현재 재개 기준으로 쓰지 않는 문서다.

## 운영 규칙
- archive 이동 전 `doc/work/index.md` 또는 관련 기준 문서에 대체 위치를 남긴다.
- archive 문서는 기본적으로 수정하지 않고, 필요하면 새 버전 문서를 `doc/work/` 루트에 다시 만든다.
