# Repeato User Interview 001

## 메타
- 인터뷰 대상: 핵심 타깃 사용자(직장인, 중2 수준 영어)
- 인터뷰 일시: 2026-02-22
- 상태: Confirmed
- 목적: v1 기획/학습 루프 상세화

## 응답 요약
1. 학습 목표
- 1순위: 회화
- 다만 초기 진입은 단어 암기가 중요하다고 인식
- 학습 방향은 전문가 가이드를 따를 의향 있음

2. 현재 불편
- 외워도 빨리 잊음
- 시간 확보가 어려움
- 준비 시간이 길면 시작 전에 포기
- 즉시 시작/즉시 종료 가능한 학습 필요

3. 학습 시간 패턴
- 주 사용 패턴: 10분 x 2회
- 여유 패턴: 30분 x 2회

4. 선호 콘텐츠
- 단어 + 예문
- 예문 주제: 여행, 일상, 업무(낮은 우선순위)
- 흥미 요소: 중간중간 재미있는 과학 이야기

5. 성취 기준
- 오늘 목표 달성
- 어제보다 발전 체감

## 추가 확정 답변
- 우선순위: 여행 > 일상 > 업무
- 콘텐츠 비중: 단어 70% + 예문 30%
- 음성 기능: 후순위(초기 버전은 텍스트 중심)
- 리마인드: 짬시간 시작형 약한 리마인드 + 개인 학습 시간 예측
- 목표 단위: 시간보다 카드/양 중심 선호(최종 정책은 전문가 조정 허용)
- 시작 난이도: 중2 수준
- 언어 정책:
  - 기본 영어 중심
  - 어려운 단어/문장에는 한국어 혼합 허용 (예: i go to 학교)
- 오답 복습: 망각곡선 스케줄 기반의 "적당한 반복"
- 주말 목표: 평일과 동일

## 기획 반영 결정(v1)
1. 세션 시작 UX
- 앱 진입 후 1탭으로 학습 시작 가능
- 준비 단계 최소화(목표: 5초 이내 시작)

2. 학습 루프
- 기본 루프: 단어 카드 -> 예문 카드 -> 오답 재노출
- 비중 제어: 7:3 자동 큐

3. 콘텐츠 큐
- 주제 가중치: 여행(상) > 일상(중) > 업무(하)
- 과학 흥미 카드(짧은 trivia) 삽입 슬롯 제공

4. 목표/KPI
- 일일 기본 목표: 카드 수 기준
- 추천 시작값: 30카드(10분 x 2), 여유일 60카드(30분 x 2)
- 개선지표: 완료 카드 수, 정답률, 오답 재회상 성공률

5. 리마인드 정책
- 강한 알람 대신 약한 리마인드
- 최근 학습 시간대 기반 예측 알림(습관 형성)

## 후속 질문(다음 인터뷰)
- 여행/일상/업무 각 주제의 세부 상황 Top 10
- 한국어 혼합 허용 기준(난이도 임계치)
- 카드 목표 수(30/40/50) A/B 선호
