# Repeato User Feedback Groups v1

## 메타
- 버전: v1
- 작성일: 2026-03-08
- 상태: Active
- 목적: 반복 작업마다 사용할 사용자 그룹과 설명 방식을 정의한다.

## 1. 사용자 그룹 구성

### Group A. 짧은 시간 학습자
- 대상:
  - 5~10분 단위로 학습하는 직장인/학생
- 검증 포인트:
  - Today 즉시 시작성
  - 목표 부담감
  - 종료 타이밍의 자연스러움

### Group B. 초급 학습자
- 대상:
  - 자기평가에 자신이 낮은 초급 사용자
- 검증 포인트:
  - `알겠음/헷갈림/모르겠음` 3단계 평가 이해도
  - 뜻 보기/넘기기 동작의 명확성

### Group C. 정리 중심 사용자
- 대상:
  - 덱/카드 정리와 구조 관리에 민감한 사용자
- 검증 포인트:
  - Decks/Add의 역할 구분
  - 저장 후 복귀 흐름

### Group D. 신뢰 민감 사용자
- 대상:
  - 동기화/계정/설정에 민감한 사용자
- 검증 포인트:
  - Profile에서 무엇이 구현됐고 무엇이 아직 아닌지 이해 가능한가
  - 로컬 우선 동작이 신뢰감 있게 설명되는가

## 2. 사용자 그룹 설명 템플릿
- 이번 빌드는 무엇을 검토하는지 먼저 2문장으로 설명한다.
- 아직 구현되지 않은 항목은 "다음 반복 예정"으로 명확히 말한다.
- 피드백은 사용성, 이해도, 부담감, 다음 행동의 명확성 중심으로 받는다.

## 3. Iteration 1 설명 스크립트
- 이번 빌드는 Today 학습 흐름의 3단계 평가와 전체 탭의 역할 설명을 확인하는 버전입니다.
- 실제 동기화/결제는 아직 구현하지 않았고, 로컬 MVP를 먼저 검증하고 있습니다.
- 확인해 주실 항목:
  - Today에서 바로 시작할 수 있는지
  - `알겠음/헷갈림/모르겠음`이 자연스러운지
  - Decks/Add/Insights/Profile의 역할이 헷갈리지 않는지
