## 3차 검토 요약
1. 1차 검토(초안)
- 목적: 망각을 줄이는 반복 학습 도구
- 리스크: 과도한 알림/과금 압박/학습 피로
- 보완: 타깃 명확화, 금지사항 명문화

2. 2차 검토(충돌 조정)
- 교육전문가: 학습 목표 측정성과 난이도 곡선 강화
- 뇌과학전문가: 복습 간격/오답 재노출 근거 강화
- 보안/데이터: 과도한 데이터 수집 금지 합의

3. 3차 검토(최종 확정)
- 항목 1~6 전원 합의 확정
- 헌법 위반 가능 기능은 출시 전 차단

## 최종 결론 (항목 1~6)
1. 앱 목적
- 사용자가 "잊기 직전"에 짧게 복습해 장기 기억으로 전환하도록 돕는 일상형 기억 강화 앱을 만든다.

2. 사용자 이익
- 같은 시간 대비 더 오래 기억한다.
- 무엇을 언제 복습해야 하는지 자동 안내를 받는다.
- 여러 학습 과제를 한 앱에서 관리한다.
- 학습 진도/기억 유지 상태를 명확히 확인한다.

3. 지속 가능 포인트
- 학습 효율 개선의 체감 경험
- 짧은 성공 경험의 반복 UX
- 콘텐츠 확장 가능한 구조(과목/레벨/패키지)
- 신뢰 기반 과금(샘플 후 결제, 과잉 유도 금지)

4. 타깃 사용자
- 핵심: 학생/수험생/자격증 준비생
- 확장: 직장인(외국어/전문용어), 평생학습자
- 맥락: 하루 5~20분 짧은 빈 시간 학습

5. 필수 철학/핵심 기능
- 철학:
  - 과학적 근거 기반 복습(망각곡선 + 인출 연습)
  - 사용자 신뢰 우선(데이터/과금/알림 절제)
  - 짧고 꾸준한 반복
- 핵심 기능:
  - 카드형 학습 + 빠른 정오답 입력
  - 망각곡선 기반 복습 스케줄 자동 생성
  - 오답/취약 카드 재노출 규칙
  - 과제별 진도/기억 유지 대시보드
  - 샘플 체험 후 결제 가능한 콘텐츠 구조

6. 하면 안 되는 부분
- 근거 없는 자극형 알림/중독형 리텐션 설계
- 과도한 과금 유도(핵심 복습 기능 잠금, 다크패턴)
- 개인정보 과수집/목적 외 사용
- 학습 목표와 무관한 과도한 소셜/게임화
- 검증 없는 복습 정책 임의 변경
