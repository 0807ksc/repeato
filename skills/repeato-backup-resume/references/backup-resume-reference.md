# Backup Resume Reference

## Snapshot Fields
- Current state summary
- Completed work in this iteration
- Current blocked items or cautions
- Next priority candidates
- Resume starting steps
- Required reference docs
- Git and GitHub status
- User decisions still needed

## Backup Order
1. Summarize the current iteration.
2. Record blocked items and unresolved decisions.
3. List the next 1 to 3 actions.
4. Capture restart steps.
5. Save the snapshot to `doc/work/`.
6. Reflect the same state in `doc/context/context-log.md` and `doc/context/next-actions.md`.

## Resume Order
1. Read the latest snapshot.
2. Read `doc/context/next-actions.md`.
3. Read `doc/context/context-log.md`.
4. Check current branch and git status.
5. Check GitHub auth, issue, and PR state if relevant.
6. Pick one next action and continue.

- Template: `doc/work/repeato-resume-snapshot-template-v1.md`
- Mutable context: `doc/context/context-log.md`, `doc/context/next-actions.md`
